Gopher Express
=========================

If your're in Glitch, just "remix".
To run locally: 
 * ```npm install```, then ```npm start```
 * map a domain to localhost with [ngrok](http://ngrok.io)
 * create a new extension on gopher.email with that domain
 * copy config.example.js to config.js
