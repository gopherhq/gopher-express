Extension Settings

Your extension settings are are saved in the core Gopher API, allowing your extension to be wonderfully stateless. These settings are sent with with every Gopher webhook under the key: extension.private_data.

Store useful things! For example:

  • Custom followup messages
  • User logos
  • OAuth Tokens granting access to antoher service

Adding and removing settings is easy, just copy / paste form fields (view source). Your extension stores free-form JSON so new settings Just Work©.

This framework logs all settings to the console. This is the same data that will arrive in the webhooks.

Please note that this is an alpha-release. Contributions, bug-fixes and feedback are welcome.
