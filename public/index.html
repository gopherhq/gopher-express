Settings

Extension settings are are saved in the core Gopher API (already connected), allowing your extension to be wonderfully stateless. These settings are sent with with every Gopher webhook. You may want to store:

  • OAuth Tokens granting access to external services
  • Custom followup messages
  • User logos
